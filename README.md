# ubjson-benchmark
A quick benchmark to test which is the fastest and the lightest.
The goal being to find the best transport protocol for the Kalm framework.

## Results

- Ubuntu 15.10
- Node 5.0.0
- e5-1650 @ 3.20

### JSON 

* speed: 369 ms
* size: 144 bytes

### UBJSON

* speed 2713 ms
* size 11 bytes
